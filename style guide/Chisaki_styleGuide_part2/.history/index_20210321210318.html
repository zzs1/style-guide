Color theme

Our main color theme is based on blues which make people calm

Since our web app is about mental health disorder, we decided to use the color which makes people calm and relax.
In many hospitals, color of greens and blues are used as subcolor theme due to its power of making poeple calm.
Therefore, we selected blues as main theme, and we also use grey scale tones.

Black:#000000 color is only used for showing the percentage of having mental disorder
Cultured:#F1EFEF color is for hovering effect color. WHen user click to see more infomation, the background will be less bright by adding the color.
Gainsboro:#DADADA color is used for unclicked button color as well as some background color.
White:#FFFFFF color is mainly used for entire background color or text color with the background of "Middle Blue Green". It is also used for icon colors.
Maximum blue:#52A1C3 color is used for clicked button color. This color is also used for enphasis the text color.
Middle Blue Green:#82DED9 color is our main theme color in the app. This color will appear as some background color as well as the buttons color

Typography

We mainly use "Seria Semi Condensed" font for providing the infomation to our web app users.
