Typography choice

Seria Semi Condensed and Roboto fonts are used in our web application

Seria Semi Condensed is the main font we implemented in our design
The font itself has rounded edge, it is clean to look at and less edgy then other typography we encounted
Roboto Font is the second choice of font, Roboto font format a little bit loose then the first font. therefore it is a good choice on question section since users would expect to read every single character through
