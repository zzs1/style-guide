Color theme

Our main color theme is based on blues which make people calm

Since our web app is about mental health disorders, we decided to use color which makes people calm and relax.
In many hospitals, the color of greens and blues are used as a sub-color theme due to their power of making people calm.
Therefore, we selected blue as the main theme, and we also use greyscale tones.

Black:#000000 color is only used for showing the percentage of having a mental disorder.

Black:#000000 color is only used for showing the percentage of having a mental disorder
Cultured:#F1EFEF color is for hovering effect color. When the user clicks to see more information, the background will be less bright.
Gainsboro:#DADADA color is used for unclicked button color as well as some background-color.
White:#FFFFFF color is mainly used for the entire background color or text color with the background of "Middle Blue Green". It is also used for icon colors.
Maximum blue:#52A1C3 color is used for clicked button color. This color is also used for emphasizing the text color.
Middle Blue Green:#82DED9 color is our main theme color in the app. This color will appear as some background color as well as the color of the button

Typography

Seria Semi Condensed and Roboto fonts are used in our web application

We mainly use "Seria Semi Condensed" font for providing the information to our web app users.
The font is easy to read yet also be able to hook users' attention which allows increasing the possibility to let the users read our information without being bored.
Also, we use "Roboto" font for asking questions to the users in the app. Moreover, we will use this font for bottons.
For quick skimming, we believe it is better to use the font that everyone familiar with. Therefore, we selected "Roboto" font for asking questions to the users.
